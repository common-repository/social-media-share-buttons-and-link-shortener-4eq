== Description ==
Social media plugin which lets you add share icons and is a powerful, extremely scalable and light weight url shortener tool introduced by 4eq.ir.
and you can add share icons for Facebook, Twitter, LinkedIn,google pluse ,telegam and whatssap, white short link.
